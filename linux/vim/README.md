# VIM

VIM 编辑器操作练习资源，按功能模块分类，适合跟练演示或自测。

| 目录 | 用途 |
|------|------|
| normal_mode/ | 普通模式操作 |
| insert_mode/ | 插入模式与补全 |
| visual_mode/ | 可视模式 |
| cmdline_mode/ | 命令行模式 |
| motions/ | 光标移动操作 |
| patterns/ | 搜索与模式匹配 |
| macros/ | 宏录制与回放 |
| jumps/ | 跳转列表 |
| files/ | 文件管理与 buffer |
| quickfix/ | 编译与快速修复 |
| search/ | 搜索技巧 |
| substitution/ | 替换操作 |
| copy_and_paste/ | 复制粘贴与寄存器 |
| global/ | :global 命令 |
| ctags/ | 标签跳转 |
| customizations/ | VIM 配置定制 |
| spell_check/ | 拼写检查 |
| the_vim_way/ | VIM 思维模式 |
| grep/ | grep 集成 |
| ex_mode/ | Ex 模式 |
| auto_complete/ | 自动补全（含 webapp） |
